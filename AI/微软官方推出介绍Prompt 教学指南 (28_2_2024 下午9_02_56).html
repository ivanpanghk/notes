Prompt 可以说是使用GPT的关键，在引导产生内容方面有重大影响，如何选择和使用 Prompt 是我们需要关注的问题。

最近微软官方推出一份完整使用指南，该内容介绍了 Prompt - 提示设计和工程中的一些高级技术，涵盖系统消息、小样本学习、非聊天场景等等内容。

01 指南官方链接

Microsoft Prompt engineering techniques

https://learn.microsoft.com/en-us/azure/cognitive-services/openai/concepts/advanced-prompt-engineering?pivots=programming-language-chat-completions

指南介绍了Prompt - 提示设计和提示工程中的一些高级技术。为初学者提供了快速工程指南的介绍。虽然提示工程的原则可以推广到许多不同的模型类型，但某些模型需要专门的提示结构。

目前有两种不同的 API 可以发挥提示工程的作用：Chat Completion API 和 Completion API。

提供了系统消息和小样本学习的示例。

它还解释了信息在提示中出现的顺序很重要，并且在提示开始时告诉模型您希望它执行的任务，然后再共享其他上下文信息或示例，这有助于产生更高质量的输出。

03 指南第一部分
System message - 系统讯息
系统讯息包含在 Prompt 的开头，用于为模型提供上下文、说明或与用例相关的其他资讯。使用者可以透过系统讯息描述模型应该回答什么、不应该回答什么，以及定义模型回复的格式。
