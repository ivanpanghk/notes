简介: Use AI to translate code from one language to another. 

大家工作中是否遇到过这种需求：需要将一个程序语言转为另外一种程序语言，比如将Python转为java?

通常的做法都是需要开发首先分析实现，然后再用新的开发语言重新实现。今天给大家推荐一款AI开发语言翻译应用，可以让AI帮助我们实现语言转换的需求。

  使用效果

UI界面相当的干净整洁，使用也非常的简单，只需要三步，就能实现开发语言的翻译：
第一步：输入需要翻译的代码，并选择语言类型
第二步：选择输出的语言类型
第三步：点击“Translate”

当然，在开始之前，还需要选择使用的AI模型，比如选择GPT-3.5。


  部署

部署流程和UI界面一样简洁，没有额外多余的配置，下载即用。

首先需要将代码从github上下载到本地

git clone https://github.com/mckaywrigley/ai-code-translator.git

安装相关的依赖

npm i

接下来就是可以在本地运行了，体验AI翻译开发语言的服务了。

npm run dev
