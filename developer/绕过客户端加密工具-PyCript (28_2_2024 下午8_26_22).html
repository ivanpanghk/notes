1、简介

Pycript 是一个 Burp Suite 扩展，使用户能够加密和解密请求和响应，以进行手动和自动应用程序渗透测试。它还允许用户使用JavaScript，Python和Java创建自定义加密和解密逻辑，从而可以根据特定需求量身定制加密/解密过程。

2、内容

PyCript提供了使用三种不同编程语言实现自定义加密和解密算法的灵活性。要使用PyCript，必须在用户或系统环境中至少安装并正确配置其中一种语言，以便能够从系统内的任何位置进行访问。

Python or Java or NodeJS + NPM使用Jython配置的Burp Suite用于安装Python扩展

安装

从github下载PyCript

下载地址：https://github.com/Anof-cyber/PyCript

下载文件ZIP文件后解压。

然后打开burpsuite进行扩展安装。

导入py文件后，扩展安装成功。

使用

具体使用可自行查看以下链接

https://pycript.souravkalal.tech/0.2/

其中对插件的功能模块还是挺详细的。
